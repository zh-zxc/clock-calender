# 数字时钟与日历记事本

一个功能丰富的网页应用，包含实时数字时钟、日历和记事本功能。

## 功能特性

- **实时数字时钟**：显示当前时间（小时:分钟:秒）
- **日历组件**：可查看任意月份的日历，支持月份切换
- **记事本功能**：为特定日期添加、查看和删除记事
- **主题切换**：
  - 深色模式：黑色星空背景，适合夜间使用
  - 浅色模式：蓝天白云背景，适合白天使用
- **响应式设计**：在手机、平板和桌面设备上都能良好显示
- **数据持久化**：使用浏览器localStorage保存记事数据

## 技术实现

- 纯前端实现（HTML, CSS, JavaScript）
- 模块化架构设计，状态管理和UI控制分离
- 使用现代CSS特性（渐变、动画、flexbox等）
- 无外部依赖库

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 或使用本地服务器运行（例如Python的http.server）

### 使用Python运行：

```bash
python3 server.py
```

然后在浏览器中访问 `http://localhost:3000`

## 主题切换

- 点击右上角的"切换到浅色模式"/"切换到深色模式"按钮切换主题
- 深色模式：黑色背景配闪烁星空
- 浅色模式：蓝天白云背景

## 功能操作

1. **日历操作**：
   - 点击"< 上月"和"下月 >"按钮切换月份
   - 点击具体日期可在记事本中查看或添加该日期的记事

2. **记事本操作**：
   - 在文本框中输入记事内容
   - 点击"保存记事"按钮保存
   - 点击记事旁的"✕"按钮删除记事
