# 数字时钟与日历记事本

一个功能丰富的网页应用，包含实时数字时钟、日历和记事本功能。

## 功能特性

- **实时数字时钟**：显示当前时间（小时:分钟:秒），作为界面核心元素突出显示
- **日历组件**：可查看任意月份的日历，支持月份切换，采用紧凑设计减少占位
- **记事本功能**：为特定日期添加、查看和删除记事
- **消费记录功能**：记录和查看特定日期的消费情况，包含统计信息
- **主题切换**：
  - 深色模式：黑色星空背景，适合夜间使用
  - 浅色模式：蓝天白云背景，适合白天使用
- **响应式设计**：在手机、平板和桌面设备上都能良好显示
- **数据持久化**：使用浏览器localStorage保存记事和消费数据
- **增强交互体验**：按钮具有水波动画和悬停效果，提升用户操作感知

## 技术实现

- 纯前端实现（HTML, CSS, JavaScript）
- 模块化架构设计，状态管理和UI控制分离
- 使用现代CSS特性（渐变、动画、flexbox等）
- 无外部依赖库

## UI设计亮点

- **信息层级优化**：时间作为核心信息置于界面顶部中央，增大显示尺寸以突出重要性
- **布局重构**：日历组件采用紧凑设计并放置在侧边，减少视觉占位
- **功能整合**：记事和消费功能模块紧随时间显示下方，形成逻辑连贯的信息流
- **交互增强**：所有可交互元素均提供视觉反馈，包括悬停效果、水波动画等

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 或使用本地服务器运行

### 使用Python运行：

```bash
python3 server.py
```

然后在浏览器中访问 `http://localhost:3000`

### 使用Node.js运行：

```bash
node server.js
```

或者使用npm命令：

```bash
npm start
```

然后在浏览器中访问 `http://localhost:3000`

## 主题切换

- 点击右上角的"切换到浅色模式"/"切换到深色模式"按钮切换主题
- 深色模式：黑色背景配闪烁星空
- 浅色模式：蓝天白云背景

## 功能操作

1. **时钟**：
   - 实时显示当前时间，每秒自动更新

2. **日历操作**：
   - 点击"< 上月"和"下月 >"按钮切换月份
   - 点击具体日期可在记事本和消费记录中查看或添加该日期的相关内容
   - 日期右下角的标记点表示该日期有记事（蓝色）或消费记录（粉色）

3. **记事本操作**：
   - 在文本框中输入记事内容
   - 点击"保存记事"按钮保存
   - 点击记事旁的"✕"按钮删除记事

4. **消费记录操作**：
   - 输入消费金额和描述
   - 点击"记录消费"按钮保存
   - 点击消费记录旁的"✕"按钮删除记录
   - 底部显示当日、当月消费统计信息
