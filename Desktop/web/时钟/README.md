# 数字时钟应用

一个简洁的数字时钟网页应用，带有本地服务器部署功能。

## 功能特点

- 实时显示小时、分钟和秒钟
- 响应式设计，在移动设备上也能良好显示
- 绿色荧光效果，具有科技感
- 可通过本地服务器运行

## 如何运行

### 方法一：使用Python（推荐）

项目使用 Python 内置的 HTTP 服务器，确保系统已安装 Python 3，然后在项目目录中执行：

```bash
python3 server.py
```

或者使用 npm 命令：

```bash
npm install
npm start
```

服务器将在 `http://localhost:3000` 启动。

### 方法二：直接打开文件

可以直接在浏览器中打开 `index.html` 文件来使用时钟应用，但这种方式无法通过网络访问。

## 停止服务器

在终端中按下 `Ctrl + C` 组合键停止服务器。
